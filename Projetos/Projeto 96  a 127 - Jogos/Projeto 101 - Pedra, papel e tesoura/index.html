Regras do Jogo

Pedra, papel e tesoura é um jogo simples de dois jogadores onde cada jogador escolhe uma das três opções ao mesmo tempo.

  • Pedra ganha de Tesoura
  • Tesoura ganha de Papel
  • Papel ganha de Pedra

Se ambos escolhem a mesma opção, o jogo resultaem empate.
